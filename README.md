Курсовая работа по РЗСП. Тема:Разработка программы тестирования сетевой нагрузки Описание задачи: Разработать программный комплекс, который будет генерировать сетевой трафик на одной стороне и принимать его на другой. Собирать информацию о задержке, скорости передачи и потерях пакетов. Интерфейс генератора должен запрашивать количество пакетов, объем пакетови частоту отправки. Интерфейс клиента должен отображать прием трафика, задержу и скорость приема.
